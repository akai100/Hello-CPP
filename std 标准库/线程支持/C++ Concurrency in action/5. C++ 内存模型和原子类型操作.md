# 2. C++ 中的原子操作和原子类型

原子操作是一类不可分割的操作，当这样操作在任意线程中进行一半的时候，你是不能查看的；它的状态要不就是完成，要不就是未完成。

## 2.1 标准原子类型

标准原子类型：```std::atomic_flag```是一个简单的布尔标志，

```std::atomic<>```

## 2.2 std::atomic_flag 的相关操作

```std::atomic_flag``` 是最简单的标准原子类型，它表示了一个布尔标志。仅支持两种状态：“清除态”和“设置态”。

```std::atomic_flag``` 类型的对象必须被```ATOMIC_FLAG_INIT```初始化。初始化标志位是“清除”状态。

```c++
std::atomic_flag f = ATOMIC_FLAG_INIT;
```
