CRTP（Curiously Recurring Template Pattern）中文：奇异递归模板模式

基本形式：

```c++
template<typename Derived>
class Base {
    // 在基类中可以通过 static_cast<Derived*>(this) 调用派生类方法
};

class MyType : public Base<MyType> {
    // ...
};
```

+ 派生类将自己作为模板参数传给基类；

+ 基类在编译期就知道派生类类型；

+ 无需虚函数，实现零开销多态；

## 核心用途

| 用途 | 说明 |
|------|------|
| 静态多态 | 在基类中调用派生类的实现，无虚函数开销 |
| 自动注入通用功能 | 如比较运算符、序列化、引用计数等 |
| 强制接口实现 | 通过基类要求派生类提供特定方法 |
