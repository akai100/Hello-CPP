+ 第一步：调用 p 所指向对象的析构函数（```~T()```）；

+ 第二步：：调用 ```operator delete``` 函数（通常是全局的）释放内存


📌 补充：

如果使用的是 placement new（在指定地址构造对象），则不能用 ```delete```，而应手动调用析构函数，因为内存不是通过 new 分配的：

```c++
char buffer[sizeof(MyClass)];
MyClass* obj = new(buffer) MyClass(); // placement new
obj->~MyClass(); // ✅ 正确：只调析构
// ❌ 错误：delete obj; （会尝试释放栈内存！）
```
