## 定义

+ 通用引用 是指形如 ```T&&``` 的声明，且 T 是一个模板参数，并处于类型推导上下文中

+ 它可以绑定到 左值 或 右值，具体类型由实参决定

| 写法 | 上下文 | 类型 |
|------|--------|------|
| `int&& r = 42;` | 非模板，`int` 是具体类型 | 右值引用（只能绑定右值） |
| `template<typename T> void f(T&& x);` | 模板，T 被推导 | 通用引用（可绑定左值或右值） |

```c++
void g(int&& x); // x 是右值引用

template<typename T>
void f(T&& x);   // x 是通用引用！

int main() {
    int a = 10;
    f(a);     // T 被推导为 int& → x 是 int&（左值）
    f(42);    // T 被推导为 int  → x 是 int&&（右值）
}
```

📌 关键规则（引用折叠）：

+ ```T& &&``` → ```T&```

+ ```T&& &&``` → ```T&&```
  
编译器通过引用折叠，让 T&& 在模板中能“自适应”左值/右值。
