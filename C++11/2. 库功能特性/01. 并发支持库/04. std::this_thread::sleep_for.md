
```std::this_thread::sleep_for``` 是 C++11 引入的线程休眠函数，用于让当前线程阻塞（挂起）一段指定的时长。


## 1. 基础用法与签名

+ 头文件：```<thread>``` 和 ```<chrono>```

+ 签名

```c++
template< class Rep, class Period >
void sleep_for( const std::chrono::duration<Rep, Period>& sleep_duration );
```

+ 行为

  + 当前线程进入 Blocked (阻塞) 状态
 
  + 不占用 CPU 时间片（这是与 yield 的最大区别）
 
  + 至少休眠指定的时间，但实际休眠时间可能更长（取决于操作系统调度器的精度和负载）
 

+ 代码示例

```c++
// 休眠 500 毫秒
std::this_thread::sleep_for(500ms); 
    
// 或者显式构造
std::this_thread::sleep_for(std::chrono::milliseconds(500));
```
