# 列表初始化

从**花括号初始化器列表**初始化1对象

+ 直接列表初始化
  ```C++
  T 对象{实参1, 实参2, ...};
  T{实参1, 实参2, ...}
  new T(实参1, 实参2, ...);
  类 { T 成员{实参1, 实参2, ...}; };
  类::类() : 成员{实参1, 实参2, ...} {}
  ```

+ 复制列表初始化
  ```C++
  T 对象 = {实参1, 实参2, ...};
  
  函数({实参1, 实参2, ...})

  return {实参1, 实参2, ...};

  对象[ {实参1, 实参2, ...} ]

  对象 = {实参1, 实参2, ...};

  U( {实参1, 实参2, ...} )

  类 { T 成员={实参1, 实参2, ...}; };
  ```

  为类型定义列表初始化
  ```C++
  struct Q {
      Q() = default;
      Q(Q const&) = default;
      Q(std::initializer_list<Q>) {}
  };

  Q q;
  Q q2 = Q{q};
  ```
