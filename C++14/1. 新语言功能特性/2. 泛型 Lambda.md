
在 C++14 中，泛型 Lambda (Generic Lambda) 是一项极具革命性的特性。它允许你在 Lambda 表达式的**参数列表**中使用 ```auto``` 关键字，从而使 Lambda 函数能够接受任意类型的参数，而无需预先指定具体类型。

## 1. 核心语法

```c++
// C++11: 只能处理 int
auto add = [](int a, int b) { return a + b; };
```

在 C++14 中，可以使用 ```auto```：

```c++
// C++14: 可以处理任何支持 + 操作的类型
auto add = [](auto a, auto b) { return a + b; };
```

## 2. 底层原理

当你编写一个泛型 Lambda 时：

```c++
auto lambda = [](auto x, auto y) { return x + y; };
```

编译器实际上生成了类似下面的一个匿名类（伪代码）：

```c++
class __AnonymousClosure__ {
public:
    // 注意：operator() 变成了一个模板函数
    template <typename T1, typename T2>
    auto operator()(T1 x, T2 y) const {
        return x + y;
    }
};
```
