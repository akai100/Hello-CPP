## 1. 语法与用法

+ 前缀：0b 或 0B（推荐用小写 0b，更易读）。

+ 字符：只能包含 0 和 1。

+ 类型：默认为 int，如果数值过大，遵循整型字面量的后缀规则（如 u, l, ll）。

## 2. 代码示例

```c++
// C++14 之前：必须用十六进制或十进制脑补
// int mask = 0x0F; // 0000 1111
    
// C++14：直接写二进制，直观清晰
int mask_low = 0b00001111; 
int mask_high = 0b11110000;
    
// 结合 ' 分隔符 (也是 C++14 特性) 提高可读性
int byte_val = 0b1010'1010; 
    
// 带后缀
unsigned long long big_bin = 0b1111'u64;
```
