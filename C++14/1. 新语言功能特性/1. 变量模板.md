变量模板 (Variable Templates) 是一项非常重要的新特性。它允许你为变量定义模板，从而可以为不同的类型生成不同版本的变量（通常是常量）。


在此之前（C++11及以前），模板只能用于类 (class/struct) 和 函数 (function)。如果你想为不同类型定义一个数学常数（如 π），你不得不为每个类型写一个特化，或者使用复杂的 struct 包装。

## 1. 基本语法

变量模板的声明非常简单，只需在变量声明前加上 ```template <typename T>```：

```c++
template <typename T>
constexpr T pi = T(3.1415926535897932385);
```

## 2. C++11 VS C++14 对比

### 2.1. C++11 写法
```c++
// C++11 风格
template <typename T>
struct Pi {
    static constexpr T value = T(3.1415926535897932385);
};

// 使用
double d = Pi<double>::value;
float f = Pi<float>::value;
```

### 2.2. C++14 写法

```c++
// C++14 风格
template <typename T>
constexpr T pi = T(3.1415926535897932385);

// 使用
double d = pi<double>;
float f = pi<float>;
auto a = pi<long double>;
```
