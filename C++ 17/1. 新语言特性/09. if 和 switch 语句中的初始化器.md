**C++17 之前：**

```if``` 和 ```switch``` 语句中如果需要局部变量，通常需要在语句块外部声明，或者将整个逻辑包裹在一个额外的作用域 ```{}``` 中。

**C++17 之后：**

C++17 引入了初始化器（Init-Statement），允许你在 ```if``` 或 ```switch`` 关键字后直接声明并初始化变量。这些变量的作用域仅限于该 ```if``` 或 ```switch``` 语句及其关联的 ```else/case``` 分支中。

## 1. 基本语法

```c++
if (初始化语句; 条件表达式) {
    // ...
} else {
    // ...
}

switch (初始化语句; 控制表达式) {
    // ...
}
```

+ 初始化语句：可以是一个变量声明（如 ```int x = 0;```）或一个表达式语句（如 ```x++;```）;

+ 分号 ```;```：必须用来分隔初始化语句和条件/控制表达式；

+ 作用域：初始化的变量生命周期从声明处开始，直到 ```if/switch``` 语句结束（包括所有的 ```else``` 或 ```case``` 分支）;
