这是 C++17 引入的一项语法改进，旨在减少代码冗余，特别是在需要从同一个命名空间引入多个标识符时。

## 1. 语法变化

### 1.1 C++ 17 以前

你需要为每个名称单独写一行 using 声明：

```c++
namespace std {
    // 假设我们要引入这些
}

using std::vector;
using std::string;
using std::cout;
using std::endl;
```

### 1.2 C++17

你可以使用逗号分隔列表在一条语句中引入多个名称：

```c++
using std::vector, std::string, std::cout, std::endl;
```
