## 1. ```std::apply```

```std::apply``` 是 C++17 引入的一个极其实用的工具函数，位于 ```<tuple>``` 头文件中。

它的核心作用是：将一个元组（```std::tuple```、```std::pair``` 或任何支持 ```std::get``` 和 ```std::tuple_size`` 的类）“解包”（unpack），并将其中的元素作为参数传递给一个可调用对象（函数、Lambda、函数对象等）。

### 1.1 函数签名

```c++
template< class F, class Tuple >
constexpr decltype(auto) apply( F&& f, Tuple&& t );
```

+ ```f```: 可调用对象（函数指针、Lambda、std::function、成员函数指针包装器等）

+ ```t```:  元组对象（如 std::tuple, std::pair）

+ 返回值: ```f``` 被调用后的返回值

逻辑等价于：

如果元组 ```t``` 包含元素 ```e1, e2, ..., eN```, 那么：

```c++
std::apply(f, t) 
// 等价于 
f(std::get<0>(t), std::get<1>(t), ..., std::get<N-1>(t))
```

### 1.2 基本使用示例

```c++
void print_info(int id, const std::string& name, double score) {
    std::cout << "ID: " << id 
              << ", Name: " << name 
              << ", Score: " << score << "\n";
}

 auto data = std::make_tuple(101, "Alice", 95.5);

// C++17 之前：需要手动 get<0>, get<1>, get<2> 或者写复杂的展开模板
// C++17: 一行搞定
std::apply(print_info, data);
```

### 1.3 高级应用场景

```std::apply``` 完美支持完美转发（Perfect Forwarding）。如果你在一个模板函数中接收了一个元组，并希望将其内容转发给另一个函数：

```c++
template<typename Func, typename Tuple>
void wrapper(Func&& f, Tuple&& t) {
    // 完美转发元组内容给函数 f
    std::apply(std::forward<Func>(f), std::forward<Tuple>(t));
}

void target(int a, double b) { /* ... */ }

auto t = std::make_tuple(1, 2.5);
wrapper(target, t); 

```
