## 1. 功能

```std::as_const``` 是一个辅助函数模板，位于 ```<utility>``` 头文件中。它接受一个左值引用 ```T&```，并返回一个 ```const T&```

## 2. 主要用途

1. 强制调用 ```const``` 重载函数：当你有一个非 ```const``` 对象，但想调用它的 ```const``` 成员函数版本时

2. 范围 ```for``` 循环中避免拷贝：当容器很大且你只需要读取时，使用 ```as_const``` 可以确保循环变量是 ```const``` 引用，防止意外修改并可能触发移动语义（如果容器实现了特定的优化）

## 3. 使用示例

```c++
 MyClass obj;
    
// 1. 强制调用 const 版本
// 如果没有 as_const，obj.print() 会调用非 const 版本
std::as_const(obj).print(); // 输出: Const version called.

// 2. 在范围 for 循环中使用
std::vector<std::string> words = {"hello", "world"};
    
// 错误示范：如果不小心写了 auto word : words，会发生拷贝
// 正确示范：使用 as_const 确保获取的是 const 引用，且语义明确为“只读”
for (const auto& word : std::as_const(words)) {
    // word 的类型是 const std::string&
    std::cout << word << " ";
    // word += "!"; // 编译错误：不能修改 const 对象
}
std::cout << std::endl;
```
