
```std::variant``` 是 C++17 标准引入的一个模板类，位于 ```<variant>``` 头文件中。它可以被理解为一个类型安全的联合体（type-safe union）。

与传统的 C/C++ union 不同，```std::variant``` 在任何时刻只存储其备选类型列表中的一种类型的值，并且它会记录当前存储的是哪种类型，从而避免了读取错误类型导致的未定义行为。

## 1. 核心特性

+ 类型安全

  编译器会检查访问操作是否合法。如果尝试获取当前未存储的类型，会抛出 ```std::bad_variant_access``` 异常（使用 ```std::get```）或返回编译错误（在某些上下文中）;

+ 值语义

  支持拷贝构造、移动构造、赋值等操作，管理内部对象的生命周期（自动调用析构函数）;

+ 非空性

  ```std::variant``` 必须始终持有某个类型的值（除非该类型本身允许为空，如指针或 ```std::optional```）。默认构造时，它会初始化列表中的第一个类型；

+ 替代 ```union``` 和继承多态

  常用于实现需要存储多种不同类型数据的场景，是比 ```union``` 更安全、比继承层次结构更轻量（无虚函数表开销）的替代方案

## 2. 基本用法

## 2.1. 定义与初始化

```c++
// 定义一个可以存储 int, double, 或 std::string 的 variant
using MyVariant = std::variant<int, double, std::string>;

// 默认构造：初始化为第一个类型 (int)，值为 0
MyVariant v1; 

// 直接初始化为 double
MyVariant v2 = 3.14; 

// 显式构造为 string
MyVariant v3 = std::string("Hello");

// 或者使用 std::in_place_index 或 std::in_place_type
MyVariant v4(std::in_place_type<std::string>, "World"); 
```

## 2.2. 访问值

主要有三种方式访问 ```std::variant``` 中的值：

+ ```std::get<T>(v)``` 或 ```std::get<I>(v)```：

  + 通过类型 ```T``` 或索引 ```I``` 获取值；

  + 如果当前存储的类型不匹配，会抛出 ```std::bad_variant_access``` 异常；

+ ```std::get_if<T>(&v)``` 或 ```std::get_if<I>(&v)```

  + 返回指向值的指针
 
  + 如果类型不匹配，返回 ```nullptr```，不会抛出异常，适合用于不确定类型的场景

+ ```std::visit``` (推荐)：

  + 这是处理 ```std::variant``` 最强大且安全的方式。它接受一个访问者（通常是 ```lambda``` 表达式或函数对象）和 ```variant``` 对象；
 
  + 访问者需要为 ```variant``` 可能的每种类型都提供重载（或使用泛型 lambda）;
 
  + ```std::visit``` 会根据 ```variant``` 当前实际存储的类型，自动调用对应的重载函数；

## 3.3 检查当前类型

+ ```v.index()```：返回当前存储类型的索引（从 0 开始）。如果 ```variant`` 处于无效状态（极少见，通常在异常发生后），返回 ```std::variant_npos```；

+ ```std::holds_alternative<T>(v)```：布尔函数，检查 ```variant``` 当前是否持有类型 ```T```；

## 3.4. 使用示例

```c++
using Var = std::variant<int, double, std::string>;
std::vector<Var> data = {42, 3.14159, std::string("C++17")};

// 定义一个访问者 (lambda)
auto visitor = [](const auto& value) {
// decltype(value) 会推导出当前访问的具体类型
if constexpr (std::is_same_v<decltype(value), const int&>) {
    std::cout << "Integer: " << value << "\n";
} else if constexpr (std::is_same_v<decltype(value), const double&>) {
    std::cout << "Double: " << value << "\n";
} else if constexpr (std::is_same_v<decltype(value), const std::string&>) {
    std::cout << "String: " << value << "\n";
}
};

// 或者使用重载集 (C++17 常用技巧)
struct Overloaded {
    template<typename T> void operator()(const T& val) {
        std::cout << "Generic: " << val << "\n";
    }
};

// 针对特定类型的特化
struct SpecificVisitor {
    void operator()(int i) { std::cout << "Int: " << i << '\n'; }
    void operator()(double d) { std::cout << "Double: " << d << '\n'; }
    void operator()(const std::string& s) { std::cout << "String: " << s << '\n'; }
};

for (const auto& v : data) {
    // 使用 std::visit 调用访问者
    std::visit(SpecificVisitor{}, v);
        
    // 也可以使用 lambda 直接作为访问者
    /*
        std::visit([](const auto& arg) {
            std::cout << "Value: " << arg << ", Type index: " << /* 需要在外部获取 index 或使用 if constexpr * / "\n";
        }, v);
    */
}

// 演示 get_if 的安全访问
Var myVar = 100;
if (int* ptr = std::get_if<int>(&myVar)) {
    std::cout << "Got int safely: " << *ptr << std::endl;
} else {
    std::cout << "Not an int." << std::endl;
}
```
