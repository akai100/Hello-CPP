```std::any`` 是 C++17 引入的一个类模板，用于安全地存储任何可拷贝构造的类型。它类似于 Python 中的 object 或 C# 中的 object，但在 C++ 中是类型安全的（需要显式转换）。

+ 头文件: #include <any>

+ 功能: 可以持有 int, double, std::string, 自定义类等任何类型的值。

+ 类型安全: 存取时必须指定正确的类型，否则抛出 std::bad_any_cast 异常。

+ 开销: 涉及动态内存分配（如果存储的对象太大，无法放入内部缓冲区）和运行时类型检查 (typeid)

## 1. 创建 any 对象，存储不同类型

```c++
 // 1. 创建 any 对象，存储不同类型
std::any a = 42;                  // 存储 int
std::any b = std::string("Hello"); // 存储 string
std::any c = 3.14;                // 存储 double
```

## 2. 检查类型 (typeid)

```c++
if (a.type() == typeid(int)) {
    std::cout << "a is an int\n";
}
```

## 3. 获取值

```c++
// 方式 A: 返回值拷贝
int val_a = std::any_cast<int>(a); 
    
// 方式 B: 返回引用 (避免拷贝，且可以修改)
std::string& ref_b = std::any_cast<std::string&>(b);
ref_b += " World";

// 方式 C: 指针形式 (如果类型不匹配，返回 nullptr，不抛异常)
if (auto p = std::any_cast<double>(&c)) {
    std::cout << "Value of c: " << *p << "\n";
} else {
    std::cout << "c is not a double\n";
}

```

## 4. 错误处理

```c++
try {
    // 尝试将 int 转换为 string，会抛出 std::bad_any_cast
    std::string wrong = std::any_cast<std::string>(a); 
} catch (const std::bad_any_cast& e) {
    std::cout << "Cast failed: " << e.what() << "\n";
}
```

## 5. 清空

```c++
 a.reset(); // 现在 a 为空 (has_value() 为 false)
if (!a.has_value()) {
    std::cout << "a is empty\n";
}
```

## 6. 适用场景

+ 需要在一个容器中存储完全不同类型的对象（例如 std::vector<std::any>）

+ 函数参数或返回值类型不确定（类似 void* 但更安全）

+ 脚本语言绑定或配置系统，其中值的类型在运行时才能确定
